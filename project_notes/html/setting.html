Copying file

• setting server 1. buat direktori2 yang diperlukan (conf,logs,certbot,dll)
2. download/upload filenya
3. edit file .yml nya sesuai dgn path dan kebutuhannya
4. docker save myimage:latest | gzip > myimage_latest.tar.gz = save image dari sserver lain
5. download file dfpy nya dari server lain
6. docker load -i (nama image).tar.gz = pasang image yang sudah di download
7. buat file default.conf dengan acuan file yang ada di local
8. pindahkan defualt.conf ke folder conf
9. edit docker-compose.yml tambahkan service certbot
10. curl -L https://raw.githubusercontent.com/wmnnd/nginx-certbot/master/init-letsencrypt.sh > init-letsencrypt.sh = ambil file init nya
11. edit file init dengan mengubah domain, path, email, ubah semua “docker-compose” menjadi “docker compose”
12. tutup semua server dan ssl pada file init (tanda # harus selalu di paling kanan)
13. edit file docker-compose.yml sesuaikan dengan file yg di local
14. chmod +x file init agar bisa di eksekusi
15. ./init = jalan kan filenya untuk mendapatkan certificate
16. copy certificate yang diberi
17. paste certificate nya ke default.conf
18. dan buka # server 443 dan ssl nya
19. buat file index.html di dalam folder html untuk testing
20. docker compose up -d = jalankan container
21. test di browser dengan domain yang baru
