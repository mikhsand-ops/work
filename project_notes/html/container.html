1. mkdir = buat direktori
2. nano/vim = editor
3. docker-compose.yml = file konfigurasi docker compose
4. docker ps = melihat daftar container yang sedang running
5. docker images = melihat images yang ada

8. docker compose stop = menghentikan semua kontainer yang sedang berjalan
9. docker stop (kontainer) = menghentikan kontainer tertentu yang sedang berjalan
