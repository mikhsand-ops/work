Update dist

1. buka ssh servernya
2. masuk ke direktori tujuan
3. docker ps = lihat kontainer
4. docker inspect (nginx/html) |grep working = untuk lihat kontainer yg berjalan berada di direktori mana
5. pwd = untuk melihat lokasi direktori saat ini lalu copy
6. copy file dist terbaru ke server tujuan
7. rm -rf index.html _redirects static/ dist = untuk menghapus dist yang sebelumnya
8. tar -zxf = extract file
9. mv dist/* . = pindahkan file baru nya
10. ls -lshtr = cek file apakah sudah masuk dengan diurutkan berdasarkan waktu nya
11. docker-compose stop = stop kontainer
12. docker-compose up -d = running lagi kontainernya
13. lalu cek di web nya apakah sudah terupdate versinya atau belum
