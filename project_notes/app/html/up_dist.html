Update dist

1. buka ssh servernya
2. masuk ke direktori tujuan
3. pwd = untuk melihat lokasi direktori saat ini lalu copy
4. copy file dist terbaru ke server tujuan
5. tar -zxf = extract file
6. mv dist/* . = pindahkan file nya
7. ls -lshtr = cek file apakah sudah masuk dengan diurutkan berdasarkan waktu nya
8. docker-compose stop = stop kontainer
9. docker-compose up -d = running lagi kontainernya
10. lalu cek di web nya apakah sudah terupdate versinya atau belum
